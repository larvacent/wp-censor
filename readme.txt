== Description ==

<strong>腾讯云 T-Sec 天御文本内容安全（Text Moderation System，TMS）服务使用了深度学习技术，可有效识别涉黄、涉政、涉恐等有害内容，支持用户配置词库，打击自定义的违规文本。</strong>

<strong>主要功能：</strong>

* 1、前台评论自动发送到腾讯云天御引擎进行审查，防止出现不合适的内容；

<strong>支持网站平台：</strong>

* 1. 天智软件 <a href="https://www.tintsoft.com" target="_blank" >https://www.tintsoft.com</a>


== Installation ==

* 1、把censor文件夹上传到/wp-content/plugins/目录下<br />
* 2、在后台插件列表中激活censor<br />
* 3、在《T-Sec 天御》菜单中输入腾讯云T-Sec 天御相关参数信息
